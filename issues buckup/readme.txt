Скачано с помошью плагина к хрому "Chrono Download Manager" 
(https://chrome.google.com/webstore/detail/chrono-download-manager/mciiogijehkdemklbdcbfkefimifhecn)

Файлы отобраны по маске .+/issues/\d+ на вкладке Others у его сниффера, 
который доступен через кнопку тулбара, если в выпавшем из неё окошке сверху справа нажать линзу.

issues.csv скачан в два приёма с помошью IssuesDownload-1.0-RELEASE.jar из https://github.com/psgs/IssuesDownload, 
заработавшем после установки Java SE Development Kit 8 из 
http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
